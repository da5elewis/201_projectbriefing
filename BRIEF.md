# AI Resources - Project Brief
A page to collect and share links and resources related to AI.  One page, clean design, mobile-first.  I will send people here from my social profiles.
## Tech
- Pure HTML, CSS, and JavaScript - no frameworks, no build tools
Single index.html file (CSS and JS can be separate files)
Deploys to Vercel as a static site

## Design
- Centered card layout, max-width 480px
- "AI Resources" as a heading, short site description underneath
- Profile photo placeholder (circle, 120px)
- Dark mode by default with a light/dark toggle
- Smooth hover animations on the link buttons
- Google Font: Inter

## Links
Display these as stacked buttons, full-width within the card:
- What AI Can Teach Us About Designing Better KPIs (https://sloanreview.mit.edu/article/what-ai-can-teach-us-about-designing-better-kpis)
- How Successful Sales Teams Are Embracing Agentic AI (https://hbr.org/2025/09/how-successful-sales-teams-are-embracing-agentic-ai)
- From Headlines to Digests: How Agents Personalize the Firehose (https://hackernoon.com/from-headlines-to-digests-how-agents-personalize-the-firehose)

## Nice to have
- Link cards have a small (120px-120px) thumbnail image
- Link buttons have icons (use simple SVG or emoji)
